# 🛍️ Smart Retail Analytics & Decision Support System

> An end-to-end Retail Business Intelligence solution developed using **Python, SQL, and Power BI** to analyze sales performance, customer behavior, inventory status, and future sales trends.

---

## 📌 Project Overview

The **Smart Retail Analytics & Decision Support System** is designed to help retail businesses make data-driven decisions by transforming raw transactional data into meaningful business insights.

The project combines **SQL for data analysis**, **Python for preprocessing and forecasting**, and **Power BI for interactive dashboards**, enabling users to monitor key performance indicators (KPIs), identify sales trends, analyze customer behavior, and improve inventory management.

# ▶️ How to Run

## Clone Repository

```bash
git clone https://github.com/your-username/Smart-Retail-Analytics-and-Decision-Support-System.git
```

## Install Dependencies

```bash
pip install -r Python/requirements.txt
```

## Run Python Notebooks

Open Jupyter Notebook or VS Code and execute:

- 01_data_preprocessing.ipynb
- 02_exploratory_data_analysis.ipynb
- 03_sales_forecasting_and_prediction.ipynb

Open the Power BI dashboard using:

```
Dashboard/Smart_Retail_Analytics.pbix
```
